# 日常总结 / Daily Summary

通过问答式对话帮助用户记录一天的生活。

## 功能

- 触发词：今日总结、总结今天、今天总结、每日总结、daily summary、summarize today
- 依次问四个问题：今天做了什么、今天学了什么、今天做错了什么、明天需要做什么
- 内容追加写入 `memory/日常总结.md`，按日期分隔

## 安装

```bash
npx clawhub@latest install daily-reflection
```

## 使用

说"今日总结"即可触发。

## 示例输出

```markdown
## 2026-03-19（周三）22:54

**今天做了：**
1. 下午去开会（项目交接）
2. 晚饭和朋友吃烤串

**今天学了：** 看了下 codex 的使用

**做错的：** 和朋友讨论时没有及时沟通感受

**明天要做：**
1. 看下项目代码
2. 看下需求情况

---
```
